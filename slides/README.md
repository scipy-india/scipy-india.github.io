# SciPy India Slides

This folder has the intro deck source and the compiled PDF.

If you just need the slides, use:

```bash
open slides/SciPy-intro-slides.pdf
```

If you want to rebuild them locally and have LaTeX installed:

```bash
cd slides
pdflatex Main.tex
pdflatex Main.tex
```

`Main.tex` is the entry point, and `Config.tex` holds the shared slide styling.
